### 4.4. Zooming and scrolling (if supported)

- When the user zooms in:
  - More detail becomes visible (e.g. fewer ms per division, or more bars/beats spread out).
  - The grid adjusts so that divisions remain readable.
- When the user zooms out:
  - The grid reduces its detail:
    - Minor lines may disappear.
    - Only key divisions (bars, large ms steps) remain.
